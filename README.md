# Emotion Classification from Text for Emotion-Aware Text-to-Speech

This repository contains the experiments and implementation for our group project in **42850 Natural Language Processing Algorithms – Autumn 2026**.


## Problem

Current text-to-speech systems can read text aloud, but they often use the same speaking style or tone for the entire text. This may reduce the naturalness and user experience, especially when the text contains different emotions.

Our project focuses on **emotion classification from text** as an initial NLP step toward improving text-to-speech systems. By identifying the emotional content of text segments, a future text-to-speech system could select a more suitable speaking style or voice for each part.


## Project Scope

This repository includes experiments with different NLP approaches and models for text-based emotion classification. The main goal is to compare methods and evaluate their effectiveness on an emotion classification dataset.
